# CSS 样式指令指南

## 核心原则

指令越具体，我一次做对的概率越高。以下模板可以直接套用。

---

## 1. 布局描述 — 用 ASCII 图示或明确行列

**不好的指令：** "左边是产品图片，右边是参数"
**好的指令：** "3 列等高，第 1 列产品图，第 2-3 列各有 2 排参数图"

最准确的方式是画出期望的布局：

```
桌面端：
┌──────────┬──────────┬──────────┐
│ 产品图   │ 参数1-1  │ 参数2-1  │  ← 3 列等高
│          ├──────────┼──────────┤
│          │ 参数1-2  │ 参数2-2  │
└──────────┴──────────┴──────────┘

手机端：
┌──────────┬──────────┐
│ 产品图   │ 参数1-1  │  ← 产品图左边，右列两排
│          ├──────────┤
│          │ 参数2-1  │
├──────────┼──────────┤
│ 参数1-2  │ 参数2-2  │  ← 底部两列
└──────────┴──────────┘
```

---

## 2. 分端说明 — 桌面端和手机端分开描述

每个改动请分开说明：

- **桌面端**：几列、每列几排、是否等高、间距多少
- **手机端（≤640px）**：与桌面端有什么不同
- **平板端（≤1024px）**（如果需要）：介于两者之间的变化

示例：
> 桌面端 4 列，手机端 3 列，平板保持 4 列

---

## 3. 等高一键说明

如果要求左右列等高，直接说：

> "左右两列需要等高"

我会自动用 `align-items: stretch` + `min-height: 0` 处理。

---

## 4. 颜色和文字样式

| 需求 | 指令模板 |
|------|---------|
| 文字颜色 | "文字颜色改为 `#fff`" 或 "文字改成白色" |
| 文字对齐 | "文字左对齐" / "文字居中" / "文字右对齐" |
| 文字位置 | "文字往下移一些" / "文字放在卡片底部" / "文字垂直水平居中" |
| 背景色 | "背景色改为 `var(--color-secondary)`" |
| 去掉底色 | "去掉条纹背景色" |

---

## 5. 间距速查

| 需求 | 指令模板 |
|------|---------|
| 列间距 | "列与列间距 20px" |
| 行间距 | "行与行间距 16px" |
| 统一间距 | "行列间距一致，都用 16px" |
| 减少间距 | "把间距缩小一半" |
| 版块与视频叠加 | "这个版块需要压在视频上方，轻微错位" |

---

## 6. 宽度对齐

| 需求 | 指令模板 |
|------|---------|
| 对齐导航宽度 | "内容宽度不要超过导航宽度" |
| 全宽 | "这个版块撑满全宽" |
| 使用全局宽度变量 | 我会用 `--site-width` 统一约束 |

---

## 7. 图片相关

| 需求 | 指令模板 |
|------|---------|
| 图片变形 | "图片不要变形，可以裁切" → `object-fit: cover` |
| 图片完整显示 | "图片要完整显示不能裁" → `object-fit: contain` |
| 圆角 | "卡片加圆角" 或 "圆角 12px" |
| 显示背景图 | "XX 区域要显示背景图片" |

---

## 8. 网格物品排列

| 需求 | 指令模板 |
|------|---------|
| 几个一排 | "桌面端 6 个一排，手机端 3 个一排" |
| 最后一个元素变宽 | "最后一个元素撑满剩余宽度" |
| 避免单列 | "不要出现单列行，调整列数避免" |
| 最后一行居中 | "最后一行不足时居中对齐" |

---

## 9. 完整指令示例（可直接套用）

### 示例 A：调整规格区

> 改 SPECIFICATIONS 区域：
> - 桌面端：3 列等高，产品图 1 列 + 参数 2 列各 2 排，间距 20px
> - 手机端：产品图左边占两排高度，右边两排参数，底部两列一排
> - 宽度对齐导航，图片 fill 单元格
> - 去掉参数底色，文字左对齐

### 示例 B：调整口味网格

> 改口味区：
> - 文字颜色改成白色，往下移一点
> - 桌面端 6 列一排，手机端改成 3 列一排
> - 宽度不要超过导航宽度

### 示例 C：调整特性卡片

> 改特性区：
> - 卡片稍微压在视频上方（像 AGP12000 一样的效果）
> - 中间卡片文字垂直居中，其他卡片文字在顶部
> - 手机端中间卡片占左边两排高度，右边两排，底部两列一排

## 10. 快速对照表

| 你说 | 我理解 |
|------|--------|
| "和 AGP12000 一样的效果" | 复用 AGP12000 的 CSS 模式 |
| "用全局变量" | 使用 `--site-width`、`var(--color-*)` 等 |
| "不要溢出" | 加 `min-width: 0` + `overflow: hidden` |
| "自适应" | 加 `clamp()` 或媒体查询 |
| "等高" | `align-items: stretch` + cell 高度 fix |
| "压到视频上" | `position: relative; z-index: 2; margin-top: 负值` |
| "背景铺满" | `position: absolute; inset: 0; object-fit: cover` |
| "和导航一样宽" | `width: var(--site-width); margin: 0 auto` |
| "图文分离（手机端）" | overlay `position: static; grid-area: auto; min-height: auto` |
| "图片聚焦中间" | `object-fit: cover; object-position: center` |
